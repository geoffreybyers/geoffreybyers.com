## What it is

Private dictation → transcript → draft pipeline. Records voice memos,
transcribes them, drops cleaned drafts into this repo as MDX stubs.

## Current state

Prototype works end-to-end on my machine. Publishing the first post
that was written this way is the v0 milestone.
